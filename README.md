# WikiRandom Telegram Bot

Телеграм-бот, который предоставляет случайную статью из Википедии.
Проект демонстрирует взаимодействие с публичным API (Wikipedia Action API)
и оформлен в соответствии с базовыми требованиями к чат-ботам.

## Возможности

- `/start`, `/help` — приветственное сообщение с описанием команд.
- `/wiki_random` — случайная статья из Википедии (название, краткое описание, ссылка).
- Корректная обработка ошибок (сеть, недоступность API, неверный ввод).

## Используемые технологии

- Python 3.11+
- [aiogram](https://github.com/aiogram/aiogram) 3.x — асинхронный фреймворк для Telegram Bot API
- [aiohttp](https://docs.aiohttp.org/) — асинхронные HTTP-запросы к внешнему API
- Wikipedia Action API (публичный, без ключа)
